# Python Data Analysis Project

## Project Overview
This project focuses on analyzing sales data using Python to extract meaningful insights.

## Key Tasks
- Data cleaning using pandas
- Exploratory Data Analysis (EDA)
- Basic visualizations

## Tools Used
- Python
- Pandas
- Matplotlib
- ## Dataset
The dataset contains sales transaction data including order details, customer information, and sales amount.

## Project Structure
- data/ : dataset files
- analysis.ipynb : data analysis notebook
- README.md : project documentation

## Sample Insights
- Identified top-selling products based on total sales.
- Analyzed sales trends over time.
- Highlighted customer purchase behavior.
